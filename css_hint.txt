Optional CSS tweak (recommended if shift persists):
Add to styles.css:

/* Prevent scrollbar appear/disappear causing layout shift */
html { overflow-y: scroll; scrollbar-gutter: stable; }

/* If you have a canvas wrapper, make its gutter stable too */
#canvasWrap, .canvas-wrap, #boardWrap, .board-wrap {
  scrollbar-gutter: stable both-edges;
}
